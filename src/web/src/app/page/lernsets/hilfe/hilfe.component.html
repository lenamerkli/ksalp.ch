Hilfe zum Erstellen eines Lernsets

Akzeptierte Dateiformate

1. Text

Erstellen Sie eine neue Textdatei (Windows: "Editor", macOS: "Textedit", GNU/Linux: "GEdit" oder "nano") mit der Kodierung "UTF-8" (meistens am unteren, rechten Fensterrand erkennbar) und der Endung ".txt". Zuerst wird die Frage eingetragen, danach mit einem Semikolon und einem Leerzeichen getrennt die korrekte Antwort. Pro Zeile gibt es eine Frage.

    die Medien; les médias (m.)
    ein Smartphone; un smartphone
    auf etw. verzichten; se passer de qc

Für mehrere korrekte Antworten können Sie weitere Semikolon und Leerzeichen verwenden. Die erste Antwort ist dabei die Musterlösung.

    ein(e) Jugendliche(r); un(e) ado / un(e) adolescent(e); un(e) ado; un(e) adolescent(e)

2. JSON

Erstellen Sie eine neue JSON-Datei mit der Software Ihrer Wahl. Nutzen Sie die folgende Struktur:

    [
      {
        "question": string,
        "answer": string,
        "answers": list<string>,
        "frequency": float,
        "auto_check": int
      }
      {
        "question": string,
        "answer": string,
        "answers": list<string>,
        "frequency": float,
        "auto_check": int
      }
    ]

Die Felder answers, frequency, auto_check sind optional. Aktuell wird auto_check nicht genutzt.

3. JSONL

Fast gleich wie 2. JSON, aber die Datei enthält pro Zeile ein JSON-Objekt und die Dateiendung .jsonl.

    {"question": string, "answer": string, "answers": list<string>, "frequency": float, "auto_check": int}
    {"question": string, "answer": string, "answers": list<string>, "frequency": float, "auto_check": int}

Die Felder answers, frequency, auto_check sind optional. Aktuell wird auto_check nicht genutzt.
